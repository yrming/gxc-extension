# gxc-extension

## Install

In VS Code, search for the `GXC` extension and install it.

## Usage

Use <kbd>F1</kbd> or <kbd>shift</kbd> + <kbd>command</kbd> + <kbd>p</kbd> to call out the command input, then input `GXC`.

Or Click `GXC` on the status bar.
